# MyFamilyTree
 
The FamilyTree program can be used to easily maintain and retrieve family tree information. Information saved includes: Name (Prefix, First Name, Middle Name, Last Name, Nick Name), SEX, Birthdate, Canonical Birth Date, Birth Order, Death Date, Canonical Death Date, Birth Place, Death Place, external documentation file(s), external document description, Date Added, Date Updated, etc. With one click it can link from an individualâ€™s record to the record for spouse, a parent, brothers and sisters and children.
 
It can also be used to generate HTML files that serve the same purpose and function in a similar manner.
